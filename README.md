# 🦁 Premier League Elite Valuation Tool

A Machine Learning web application that predicts the market value of football players and performs a financial ROI audit for scouts and club owners.


## 📊 Project Overview
This project bridges the gap between sports performance and financial analysis. Coming from an **accounting background**, I developed this tool to help identify "undervalued" players based on their goal output and age profile.

### Key Features:
* **Predictive Modeling:** Uses Linear Regression to estimate market value.
* **Financial Audit:** Automatically calculates if a player is "Undervalued" or "Overvalued."
* **Interactive UI:** Built with Streamlit for real-time scouting adjustments.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn, NumPy
* **Deployment:** Streamlit Cloud
* **Data Source:** Wikipedia (Web Scraped)

## 📈 How to Use
1. Adjust the **Goal Output** slider to set the player's performance.
2. Set the **Age Profile** of the player.
3. Click **"Predict Market Value"** to see the financial assessment.
